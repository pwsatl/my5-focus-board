Privacy Policy — My5 Focus Board

Last updated: July 2026

My5 Focus Board is a task management tool provided as a Chrome extension and web app for internal team use. This policy describes what data the tool handles and how.

What data is stored

  • Task data you create: task titles, notes, comments, due dates, categories, client codes, and completion history.
  • Display name (optional): a first name or nickname you choose, shown alongside your task claims, comments, and team notifications.
  • Preferences: theme, font, sound, and sync settings, including team and device sync codes.

Where it lives

All data is stored locally on your device (browser storage). If you enable team sync or device sync, your task data and display name are transmitted to and stored on a private, self-hosted Cloudflare Worker operated by the team that deploys this tool. Data is scoped by team code; only people who hold a team's code can access that team's shared lists.

What is NOT collected

  • No browsing history, page content, or activity outside the extension is read or collected.
  • No analytics, tracking, or advertising identifiers.
  • No health, financial, location, or authentication data.

Data sharing

Data is never sold or transferred to third parties. If a team configures optional notifications, task event summaries (task title, list name, and display name) are forwarded to the team's own configured webhook (e.g. Microsoft Teams via Make.com). Data is not used for any purpose beyond operating the task board.

Data removal

Removing the extension deletes all locally stored data. Synced data can be removed by the team administrator via the team's Cloudflare Worker KV storage. To request removal of synced data, contact your team administrator.

Contact

Questions about this policy can be directed to the extension publisher through the Chrome Web Store listing.
